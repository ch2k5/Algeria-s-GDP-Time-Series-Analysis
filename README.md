# Algeria GDP Forecasting using ARIMA

## Overview
This project applies **time series analysis and forecasting techniques** to model and predict Algeria's Gross Domestic Product (GDP).  
The objective is to identify an appropriate **ARIMA model** that captures the underlying dynamics of the GDP time series and produces reliable forecasts for future economic values.

Time series modeling allows us to analyze historical economic data, detect patterns such as trends and autocorrelation, and generate forecasts that can support economic analysis and decision-making.

---

## Objectives
- Analyze the historical GDP time series of Algeria.
- Identify suitable ARIMA parameters using **ACF and PACF analysis**.
- Compare competing models using **information criteria (AIC and BIC)**.
- Validate the model through **residual diagnostics**.
- Produce forecasts for future GDP values.

## Tools and Technologies
- **R**
- ARIMA Modeling
